# Household Dataset Analysis Project

## Overview

This project contains basic data analysis using Python and Jupyter Notebook. The notebook works with a household dataset and performs statistical analysis and data exploration.

## Files

maths_pr_1.ipynb
Main notebook file containing the analysis.

household_dataset_500.csv
Dataset used for the project.

## Tools Used

Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

## Analysis Included

Data loading
Data information and summary
Maximum and minimum income analysis
Range calculation
Quartile calculation
Interquartile range calculation
Basic statistical analysis

## How to Run

Install Python and Jupyter Notebook.
Install required libraries.
Open the notebook file.
Run all cells step by step.

## Purpose

The project is created for learning and practicing data analysis and statistics using Python.
